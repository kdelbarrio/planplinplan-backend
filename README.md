# Proyecto TFM - Plataforma Laravel bajo DDEV
Este repositorio contiene el entorno de desarrollo local de Laravel para el TFM, incluyendo:
- Laravel 12
- DDEV (MySQL + PhpMyAdmin)
- Configuración .env adaptada
- PhpMyAdmin accesible en https://laravel-plan.ddev.site/site:8037

## Comandos básicos
- `ddev start`: inicia el entorno
- `ddev composer install`: instala dependencias PHP
- `ddev npm run dev`: lanza Vite
- `ddev artisan migrate`: aplica migraciones
